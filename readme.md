项目名称：基于FPGA的实时FFT分析系统开发 

起止时间：2023.09~2023.12

项目介绍：在FPGA平台逻辑计算资源较少的情况下，使用流水线FFT计算结构，实现了大点数FFT的实时处理问题，在4.18W功耗下实现了16k点FFT计算，耗时189.572us。

工作概述：

1. 负责解决方案的调研：根据实时性FFT处理的需求，结合FPGA本身的可配置、并行处理的特点，查阅文献并确定使用资源消耗较少的流水线FFT计算结构。
2. 负责解决方案的具体实现和调试：负责设计各个子模块，并用verilog代码实现，产出代码3000行。
3. 负责FPGA芯片的选择：根据算法所消耗的乘法器、RAM、ROM消耗量选定了EG4X20BG256芯片。
4. 负责撰写了软件设计文档、接口文档。
5. 项目地址：https://github.com/guanjiess/fpga-fft
